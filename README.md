# UnityChanRunner

## 概要
Unityちゃんを操作して障害物を回避する3Dランゲーム

## 使用技術
- Unity 6
- C#
- Unity-chan (ユニティちゃん)

## 操作方法
| キー | 動作 |
|------|------|
| W/S | 前後移動 |
| A/D | 左右移動 |
| Space | ジャンプ |

## 開発環境
- Unity 6
- VSCode
- MacOS (Apple Silicon)
